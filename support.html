CropFocus란? / What is CropFocus?

화면의 특정 영역만 남기고 나머지를 어둡게 가려 집중을 돕는 macOS 메뉴 막대 앱입니다.

CropFocus is a macOS menu‑bar app that dims everything except a region you select, helping you focus.

자주 묻는 질문 / FAQ

단축키가 동작하지 않아요 / Shortcuts don't work

다른 앱이 같은 단축키를 사용 중일 수 있습니다. 해당 앱의 단축키를 변경한 뒤 다시 시도해 주세요.

Another app may be using the same shortcut. Change that app's shortcut and try again.

마스킹 중에도 메뉴 막대를 보고 싶어요 / Keep the menu bar visible while focusing

설정에서 "포커스 중 메뉴 막대 보기"를 켜면 됩니다.

Enable "Show menu bar while focusing" in Settings.

로그인 시 자동 실행 / Launch at login

설정에서 "로그인 시 자동 실행"을 체크하세요. (macOS 13 이상)

Check "Launch at Login" in Settings. (Requires macOS 13 or later)
